
# Customer Journey Map

## Objective
The Customer Journey Map illustrates how users interact with the OptiCrop Smart Agricultural Production Optimization Engine from accessing the application to receiving intelligent crop recommendations.

## User Persona
- Farmers
- Agricultural Researchers
- Agribusiness Organizations
- Agricultural Officers

## Journey Stages

### 1. Access the Application
- User opens the OptiCrop web application.
- The homepage displays the crop recommendation feature.

### 2. Enter Agricultural Information
The user provides:
- Nitrogen (N)
- Phosphorous (P)
- Potassium (K)
- Temperature
- Humidity
- pH Value
- Rainfall

### 3. Data Validation
The system verifies:
- Missing values
- Valid ranges
- Input format

### 4. Machine Learning Prediction
The trained model processes the data and predicts the most suitable crop.

### 5. Recommendation Display
The application displays:
- Recommended Crop
- Environmental Summary
- Prediction Result

### 6. Decision Making
The farmer uses the recommendation for cultivation planning and resource optimization.

## Benefits
- Simple user experience
- Fast prediction
- Accurate recommendations
- Improved farming decisions
