
# Data Flow Diagram (DFD)

## Overview
The Data Flow Diagram illustrates how agricultural data moves through the OptiCrop system to generate crop recommendations.

## Level 0 DFD

User
↓
Enter Soil & Climate Data
↓
OptiCrop Application
↓
Machine Learning Model
↓
Crop Recommendation
↓
Prediction Result

## Data Inputs
- Nitrogen
- Phosphorous
- Potassium
- Temperature
- Humidity
- pH
- Rainfall

## Processing
- Data Validation
- Data Preprocessing
- Feature Selection
- Model Prediction

## Outputs
- Recommended Crop
- Prediction Result
- Agricultural Recommendation

## Advantages
- Easy to understand
- Clear system workflow
- Efficient data processing
- Supports accurate prediction
